# PyVim -- The Automatic Vim Configuration System with Python

## Introduction

This is a easy tool to configurate your vim.
By default, the vim mode is Python.

## Requirement
+ colors

## Usage

To cleanup:

``` bash
./pyvim.py clean -t vimrc plugins
```

To deploy:

``` bash
./pyvim.py deploy -t vimrc pathogen
```
